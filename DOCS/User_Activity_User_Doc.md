# User Activity & Risk Overview

## 🧠 Business Context
Understanding user behavior is critical for engagement and fraud detection.

## 🎯 Objective
- Analyze user activity
- Identify high-risk users

## 🔍 Approach
Used sessions and events data with percentile-based segmentation.

## 💡 Key Insights
- Medium-risk users are most active
- High activity outliers detected

## 🚀 Business Impact
- Improved monitoring
- Better engagement strategies
